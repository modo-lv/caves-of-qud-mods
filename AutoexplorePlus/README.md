# Overview

Characters with extraordinary senses ([Heightened Hearing](https://wiki.cavesofqud.com/wiki/Heightened_Hearing), [Heightened Smell](https://wiki.cavesofqud.com/wiki/Heightened_Smell), [Electromagnetic sensor](https://wiki.cavesofqud.com/wiki/Electromagnetic_sensor)) will not interrupt autoexploring if they detect an enemy that isn't targeting them. They will, however, stop just short of melee range.

The mechanic is not tied to any specific skills, but rather the extra-sensory detection event. That means it should also work with any new senses added in the future (vanilla or modded), as long as they trigger `ExtraHostilePerceptionEvent`.

Does not work with [Penetrating radar](https://wiki.cavesofqud.com/wiki/Penetrating_radar), because that is a visibility/light extension, not an extra sense.

# Uninstall

The mod is safe to add or remove at any point.
